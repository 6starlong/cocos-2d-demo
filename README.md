# cocos-2d-demo

## 项目结构

```
cocos-2d-demo
├── assets/               # 资源目录
│   ├── animations/       # 动画剪辑 (.anim)
│   ├── audio/            # 音频资源 (.mp3, .ogg等)
│   ├── fonts/            # 字体文件 (.ttf, .otf)
│   ├── materials/        # 材质文件 (.mtl)
│   ├── models/           # 3D模型文件 (.fbx, .glTF等)
│   ├── prefabs/          # 预制体 (.prefab)
│   ├── resources/        # 需要动态加载的资源 (resources.load)
│   ├── scenes/           # 场景文件 (.scene)
│   ├── scripts/          # TypeScript 脚本 (.ts)
│   └── textures/         # 纹理贴图、图像资源
├── build/                # 构建目录
├── extensions/           # 扩展插件目录
├── library/              # 导入的资源库
├── local/                # 本地设置
├── profiles/             # 编辑器配置
├── settings/             # 项目设置
├── temp/                 # 临时文件
├── package.json          # 项目配置
├── tsconfig.json         # TypeScript 配置
└── ...                   # 其他项目文件
```
